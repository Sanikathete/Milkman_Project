# Milkman Frontend (Rajan Dairy)

React + Vite + Tailwind frontend for Rajan Dairy dairy subscription workflow.

## Setup
```bash
npm install
cp .env.example .env
npm run dev
```

## Environment
- `VITE_API_BASE_URL=http://127.0.0.1:8000/api`
- `VITE_MEDIA_BASE_URL=http://127.0.0.1:8000`

## Pages
- Home
- Login
- Register
- Product Details
- Subscription Form
- Payment Page
- Customer Dashboard
- Admin Dashboard
- Manage Products
- Manage Categories
